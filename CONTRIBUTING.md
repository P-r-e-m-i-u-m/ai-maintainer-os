# Contributing

Thanks for helping improve AI Maintainer OS.

## Development

```bash
npm install
npm run check
```

## Good Contributions

- Add a deterministic review rule
- Improve report clarity
- Add tests for edge cases
- Add safe GitHub Action behavior
- Improve maintainer documentation

## Pull Request Standard

Please include:

- What changed
- Why it matters for maintainers
- How you tested it
- Any false-positive risk

## Security

Do not add code that uploads private diffs, prompts, or repository content to third-party services by default.
