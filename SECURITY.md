# Security Policy

AI Maintainer OS inspects pull request diffs and may process sensitive source code.

## Security Principles

- No third-party upload by default
- No LLM calls in the first deterministic release
- No logging of secrets beyond file-level evidence
- PR comments should not expose secret values

## Reporting

Please report sensitive issues privately through GitHub security advisories when possible.

For public issues, avoid posting secrets, private repository content, or exploit payloads.
